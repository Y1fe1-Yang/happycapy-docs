# MCP Integration

MCP (Model Context Protocol) lets your AI agent interact with external services like GitHub, Gmail, Notion, and more.

## What is MCP?

MCP is a standardized protocol that gives AI assistants access to external tools and data sources. Think of it as "plugins" for your AI agent.

## Configuration

MCP servers are configured in `~/.mcp.json`:

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "your-token"
      }
    }
  }
}
```

## Adding New MCP Servers

You can add any MCP-compatible server. Common options:

- `@modelcontextprotocol/server-github` - GitHub
- `@codefuturist/email-mcp` - Email
- `mintlify-mcp` - Query Mintlify documentation
