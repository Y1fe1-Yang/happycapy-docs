# Agents

HappyCapy agents are powered by Claude Code and can autonomously plan, execute, and verify multi-step tasks.

## How Agents Work

```
You describe the task
       ↓
Agent plans the approach
       ↓
Agent executes step by step
       ↓
Agent verifies the result
       ↓
You get the deliverable
```

## Agent Capabilities

- **Code Generation** - Write, edit, and refactor code across any language
- **File Operations** - Create, read, edit files in the sandbox
- **Web Search** - Research topics with real-time web access
- **Tool Usage** - Run CLI tools, scripts, and build systems
- **Multi-Agent Teams** - Spawn sub-agents for parallel work

## Sandbox Environment

Each session runs in an isolated Linux sandbox with:

- 4 CPU cores, 7.8 GB RAM
- Node.js 24, Python 3.11
- Git, FFmpeg, and standard dev tools
- Full network access for API calls and web fetching
