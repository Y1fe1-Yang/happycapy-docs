# Skills

| ai-video-generation | Create videos with Veo, Seedance, Wan, and 40+ models |
| ffmpeg | Video/audio processing, format conversion, compression |
| image-enhancer | Enhance image quality, resolution, and sharpness |

## Using Skills

Skills are invoked automatically when your task matches their capability. You can also trigger them explicitly with slash commands:

```
/commit       - Create a git commit
/review-pr    - Review a pull request
```
