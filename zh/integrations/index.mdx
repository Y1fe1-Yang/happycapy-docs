HappyCapy 可以连接你常用的平台,让工作更高效。目前支持 GitHub 和 Notion 的集成,只需要简单的 Token 配置就能使用。

<Note>
**什么是 Token?** Token 就像是你给 HappyCapy 的一把钥匙,让它可以代表你访问其他平台。别担心,整个过程很简单!
</Note>
