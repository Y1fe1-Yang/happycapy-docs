HappyCapy 可以連接你常用的平台，讓工作更高效。目前支援 GitHub 和 Notion 的集成，只要簡單設定 Token 就能使用。

<Note>
**什麼是 Token？** Token 就像是你給 HappyCapy 的一把鑰匙，讓它可以代表你訪問其他平台。整個過程很簡單且安全。
</Note>
