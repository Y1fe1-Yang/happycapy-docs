# Quick Start

Get up and running with HappyCapy in just a few steps.

## Step 1: Create an Account

Sign up at [happycapy.ai](https://happycapy.ai) to get your workspace.

## Step 2: Start a Session

Once logged in, you'll have access to a fully configured sandbox environment with:

- **Claude Code** - AI coding assistant
- **96+ Skills** - Pre-built capabilities
- **MCP Servers** - External service integrations
- **Development Tools** - Node.js, Python, Git, FFmpeg, and more

## Step 3: Give Instructions

Simply describe what you want to build or accomplish. The AI agent will:

1. Plan the approach
2. Execute the steps
3. Deliver the results

<Tip>
You can use slash commands like `/commit`, `/review-pr` to trigger specific workflows.
</Tip>

## Example Tasks

```
"Build me a landing page for my product"
"Generate a video from this script"
"Analyze this dataset and create visualizations"
"Write documentation for my API"
```
