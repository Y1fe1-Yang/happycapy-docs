# Welcome to HappyCapy

HappyCapy is an AI-powered platform that lets you build, deploy, and manage intelligent agents with ease.

## What is HappyCapy?

HappyCapy provides a sandboxed environment where AI agents (powered by Claude Code) can:

- Execute complex multi-step tasks autonomously
- Use 96+ skills covering AI, design, development, and more
- Integrate with external services via MCP (Model Context Protocol)
- Generate content, code, documents, and media
